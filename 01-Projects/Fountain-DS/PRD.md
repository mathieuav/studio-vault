## Problème
Fountain a besoin de son design system traduit en composants React/Tailwind
utilisables directement par leurs développeurs.

## Objectif
Générer des composants React + Tailwind fidèles au design system FountainX Figma,
avec tokens CSS corrects, variantes complètes et documentation inline.

## Périmètre MVP
- [ ] Button (toutes variantes + états)
- [ ] Input
- [ ] Tag / Badge
- [ ] Typography scale

## Hors périmètre pour l'instant
- Animations complexes
- Storybook
- Tests unitaires

## Critères de succès
- Tokens CSS utilisés partout (zéro valeur brute)
- Toutes les variantes Figma couvertes
- Code lisible et maintenable sans connaître React
